# Customer_segmentation_system


An **AI-powered Customer Segmentation Dashboard** built using **Machine Learning and Streamlit** to analyze customer behavior and group customers into meaningful marketing segments.

This system helps businesses understand customer purchasing patterns and design **data-driven marketing strategies**.

---

## 🚀 Project Overview

Customer segmentation is an essential business strategy used to identify groups of customers based on income and spending behavior.

This project applies **K-Means Clustering** to automatically classify customers into different segments and provides actionable marketing insights through an interactive dashboard.

---

##  Objectives

* Analyze customer purchasing behavior
* Segment customers using Machine Learning
* Provide marketing intelligence insights

## 📊 Machine Learning Model

**Algorithm Used:** K-Means Clustering

**Input Features:**

* Annual Income
* Spending Score

**Output:**

* Customer Segment Classification

Segments Identified:

1. Careful Customers
2. Standard Customers
3. Target Customers
4. Careless Customers
5. Sensible Customers
